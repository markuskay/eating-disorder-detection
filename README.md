# Deep Learning for Early Detection and Prediction of Eating Disorders

A deep learning system for early detection and prediction of eating 
disorders using multi-modal clinical and behavioural data.

## Overview
This study develops and evaluates a deep learning model that integrates 
multiple data sources including clinical records, validated psychological 
assessments and anonymised behavioural indicators to detect early signs 
of eating disorders including anorexia nervosa, bulimia nervosa and 
binge-eating disorder.

## Model Architecture
- Hybrid CNN-RNN network with attention mechanisms
- Multi-modal data integration across clinical and behavioural signals
- Captures both spatial and temporal patterns in the data

## Evaluation Metrics
- Accuracy, Precision, Recall, F1-Score, AUC-ROC

## Tech Stack
- Python
- TensorFlow
- Scikit-learn
- Streamlit
- Pandas, NumPy, Matplotlib
